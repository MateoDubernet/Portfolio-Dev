Missions principales

Développement Fullstack & API : Conception de services backend et frontend robustes en Programmation Orientée Objet (POO), basés sur l'architecture MVC avec Spring Boot et Angular. Développement d'API REST testées via Postman, avec intégration des contrôles d'accès via Spring Security.
Ingénierie Frontend & UX/UI : Création d'interfaces réactives et intuitives à l'aide de la librairie Kendo UI. Implémentation de formulaires dynamiques, gestionnaires d'alertes, directives personnalisées et messages d'erreurs sur-mesure pour guider l'utilisateur.
Gestion avancée des données & Persistance : Création de requêtes SQL personnalisées (tris, filtres, modifications) et manipulation des données via l'ORM de Spring connecté à une base PostgreSQL. Développement de tableaux dynamiques avec sauvegarde de l'état des préférences utilisateur et exports automatisés vers Excel.
Sécurité & Gestion des accès : Implémentation d'un système d'authentification à double facteur (2FA). Sécurisation des comptes via la génération automatique de mots de passe forts, vérification par double saisie, masquage/affichage dynamique (bouton œil) et processus de réinitialisation sécurisé par code de validation par email.
Notifications & Temps réel : Amélioration de la réactivité opérationnelle grâce à l'intégration de Service Workers et de notifications Push pour alerter instantanément les agents de guichet de l'arrivée d'un client dans la file d'attente.
Cycle de vie & Qualité logicielle : Migration de versions d'Angular, refactorisation de code legacy et maintenance corrective. Utilisation rigoureuse des patrons de conception (Design Patterns, DTO) pour fiabiliser les flux de données.
Tests & Documentation : Écriture et exécution de tests unitaires (prioritairement sur Spring, ponctuellement sur Angular), rédaction de documentations techniques et réalisation de veilles technologiques régulières.
